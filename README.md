# Pokémon Trading Card Game 3: Grandmasters

A faithful recreation of the Game Boy Color *Pokémon Trading Card Game* series — with online multiplayer.

## 📥 Download

**[Download Latest Release](https://github.com/KaelKodes/pokemon-trading-card-game-3---Grandmasters/releases/latest)**

> Download the `.zip` for your platform, extract, and run!

## 🎮 Features

- **Full Card Pool** — 434 cards from the original GBC TCG 1 & 2 card sets
- **Battle Engine** — Complete turn-based battle system with all game mechanics (evolution, energy, status effects, trainer cards, Pokémon Powers)
- **Deck Builder** — Build and save custom 60-card decks with full validation
- **Quick Battle** — Jump into a match against an AI opponent
- **Tournament Mode** — 8-player single-elimination brackets
- **Online Multiplayer** — Real-time dueling over the internet
  - Lobby browser — create or join matches
  - Pre-match lobby with chat, deck selection, and prize voting
  - 2-minute reconnection window if you disconnect
  - Spectating support
- **Music Player** — Built-in playlist system with shuffle and volume controls
- **Premade Decks** — Browse and use premade decks filtered by energy type

## 🌐 Online Play

1. Click **Online Battle** from the main menu
2. Register a username and password
3. Create a lobby or join an existing one
4. Select your deck, vote on prize count, and ready up
5. Battle!

## 🛠️ Built With

- **[Godot Engine 4.6](https://godotengine.org/)** — Game engine
- **GDScript** — Game logic
- **Node.js + WebSocket** — Multiplayer server
- **MySQL** — Player accounts & match history

## 📋 Controls

- **Left Click** — Select cards, perform actions
- **Right Click** — Preview card details
- **Scroll** — Browse hand / bench
- **F1** - Music and Audio Controller
